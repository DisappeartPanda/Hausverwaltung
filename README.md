# ImmobilienPro

ImmobilienPro ist eine moderne Web-App für digitale Hausverwaltung mit Fokus auf Vermieter, Verwalter und Mieter.  
Die Anwendung bündelt Objektverwaltung, Wohneinheiten, Mieterkommunikation, Dokumente, Mängel, Wartungen und weitere Verwaltungsprozesse in einer klaren Plattform.

## Funktionen

- Rollenbasierte Bereiche für Vermieter/Verwalter und Mieter
- Authentifizierung mit Supabase
- Dashboard für Verwaltungsübersichten
- Verwaltung von Objekten und Wohneinheiten
- Mieterbereich mit klar getrennter Sicht
- Mängel- und Wartungsmanagement
- Dokumentenverwaltung
- Modulare Pricing-Struktur

## Tech-Stack

- Astro
- TypeScript
- Supabase
- Eigenes CSS-System mit Tokens, Base-, Components- und Theme-Dateien
